Navigating the Roadmap of Robotics

A common beginning

In robotics, different languages have different uses:

  • (1) Automation: Machine Learning: python for basic stuff, C++ for more advanced stuff
  • (2) Embedded systems and electronics: C
  • (3) Modelling: python, MATLAB

2.Development Environment

  • Linux Operating System
  • Linux Terminal
  • Git

3.Electronics

The Electronics part of Robotics can be divided into 3 parts : Microcontroller, Sensors and Actuators

4.Robot Operating System (ROS)

ROS, or Robot Operating System, is an open-source middleware framework designed to develop and control robotic systems.

5.Path Planning

Path planning is the process of “finding” a path or a trajectory for a robot by avoiding obstacles or following set dynamic or kinematic constraints of a robot.

6.Simultaneous Localization and Mapping (SLAM)

SLAM achieves is simultaneously map and localise in an environment.

7.Robot Perception

Robot Perception is something that is really underappreciated by beginners as the difficulties behind it are not properly understood

8.Machine Learning / AI

Machine Learning is about how the computer can learn to solve a task from examples, much like we humans learn.

9.control Systems

Control systems help to control the movements and functions of the robot. We need the controllers because the dynamics vary with the time.
